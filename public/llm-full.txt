SITE PURPOSE
================================================================================

Marketing funnel website for CPA (Cost Per Action) affiliate marketing education
platform. Promotes free webinars, paid mentorship programs, and educational
content. Focuses on organic traffic methods (Instagram, Telegram, content funnels)
rather than paid ads. Target audience: Indian beginners and freelancers looking
to generate online income.

================================================================================
DESIGN SYSTEM
================================================================================

Aesthetic: Pure black & white monochrome, zero border-radius throughout
Typography:
  - display-font: 'Playfair Display' (headlines, bold weights)
  - body-text: 'Source Serif 4' (body copy, readable serif)
  - label-text: 'JetBrains Mono' (uppercase metadata, 0.15em letter-spacing)
Color scheme:
  - Light: white bg (#ffffff), black fg (#000000)
  - Dark: near-black bg (#0a0a0a), off-white fg (#f5f5f5)
  - All CSS vars in app/globals.css under :root / .dark
CSS utilities:
  - btn-primary: black bg, white text (inverts in dark mode)
  - btn-secondary: transparent, black border (inverts in dark mode)
  - .card-hover: invert bg/color on hover
  - .section-rule: 4px horizontal divider
  - .noise: paper-grain SVG texture overlay
  - .lines: horizontal line pattern
  - .vertical-lines: vertical line pattern
  - .grid-pattern: 40px editorial grid
Textures use inline SVG data URIs — no external image dependencies.

COMPONENTS (components/ui/)
================================================================================

LogoLoop.tsx (client)
  - Animated marquee component (RAF-based scroll, not CSS animation)
  - Props: logos[], speed (px/s), direction (left/right/up/down),
    width, logoHeight, gap, pauseOnHover, hoverSpeed, fadeOut, fadeOutColor,
    scaleOnHover, renderItem, ariaLabel, className, style
  - Sub-components: useResizeObserver, useImageLoader, useAnimationLoop hooks
  - Per-item hover tracking via hoveredKey state (not CSS group-hover)
  - ResizeObserver for responsive sizing
  - fade-out gradient (clamp-based, CSS linear-gradient)
  - TypeScript fully typed: LogoItemNode | LogoItemImage union, LogoLoopProps

ClickSpark.tsx (client)
  - Canvas-based click spark particle effect
  - Props: sparkColor, sparkSize, sparkRadius, sparkCount, duration, easing, extraScale
  - Uses requestAnimationFrame animation loop
  - ResizeObserver for canvas sizing
  - easeFunc: ease-out (default), linear, ease-in, ease-in-out

ClickSparkProvider.tsx (client)
  - Reads theme from useTheme, passes sparkColor (white on dark, black on light)
  - Wraps children with ClickSpark component

ThemeProvider.tsx (client)
  - ThemeContext: theme (light/dark/system), resolvedTheme, setTheme, toggleTheme
  - Reads from localStorage key "voidzero-theme", falls back to prefers-color-scheme
  - Adds/removes "dark" class on document.documentElement
  - Sets colorScheme style property

ThemeToggle.tsx (client)
  - Radix DropdownMenu with Sun/Moon/Monitor icons
  - Calls setTheme("light"), setTheme("dark"), setTheme("system")
  - Shows current icon based on resolvedTheme

VercelAnalytics.tsx (client)
  - Wrapper for @vercel/analytics/react inject() call

Button (components/ui/button.tsx)
  - CVA-based variants: default, destructive, outline, secondary, ghost, link
  - Sizes: default, sm, lg, icon
  - Uses Slot from @radix-ui/react-slot
  - Does NOT use btn-primary/btn-secondary from globals.css

DropdownMenu (components/ui/dropdown-menu.tsx)
  - Full Radix DropdownMenu with: Root, Trigger, Content, Item, CheckboxItem,
    RadioItem, Label, Separator, Shortcut, Group, Portal, Sub, SubTrigger,
    SubContent
  - onPointerDown handlers in Content for close behavior

Avatar (components/ui/avatar.tsx)
  - Radix Avatar with Root, Image, Fallback

Empty (components/ui/empty.tsx)
  - Empty, EmptyHeader, EmptyMedia, EmptyTitle, EmptyDescription, EmptyContent

TextHoverEffect (components/ui/text-hover-effect.tsx) (client)
  - SVG text with radial-gradient mask reveal on hover
  - Tracks mouse position with ResizeObserver for SVG rect
  - Motion text: ghost outline stroke animation on load
  - Props: text, duration

PointerHighlight (components/ui/pointer-highlight.tsx) (client)
  - Motion div with animated border rect around children
  - Pointer SVG element
  - Uses ResizeObserver for dimensions
  - Props: children, rectangleClassName, pointerClassName, containerClassName

NotFoundPage (components/ui/not-found-page.tsx) (client)
  - Avatar "404" display
  - Empty component structure
  - Back to Home + Contact Us buttons
  - Quick links grid to all pages

IMPORTANT NOTES
================================================================================

- All pages are server components unless "use client" is declared
- termsData and privacyData are exported from their page.tsx files
- FAQ search is client-side (app/faq/page.tsx is "use client")
- Proofs page is client-side
- ThemeProvider must wrap all theme-dependent components
- ClickSpark wraps entire app — every click fires spark particles
- LogoLoop uses per-item hoveredKey state (string key `${copyIndex}-${itemIndex}`)
- fadeOut uses fadeOutColor prop — currently hardcoded rgb(250,250,250)
  (neutral-50 bg color, not dark-mode aware)
- sitemap.ts does NOT include /refund or /shipping — needs updating
- robots.ts uses hardcoded https://cpahustler.com/sitemap.xml
- No authentication, no database, no API routes
- All forms are presentational (no backend submission handlers)
- Payment processing: Razorpay (referenced in policies, not integrated)
